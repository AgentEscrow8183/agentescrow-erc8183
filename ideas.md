## Pendekatan 1: Cyberpunk Terminal / Dark Tech

**Design Movement**: Cyberpunk Neo-Noir meets Web3 Terminal

**Core Principles**:
1. Gelap dominan dengan aksen neon hijau/cyan — memancarkan aura "kode berjalan di blockchain"
2. Tipografi monospace untuk data teknis, sans-serif bold untuk heading
3. Animasi glitch dan scan-line untuk elemen interaktif
4. Layout asimetris dengan grid yang terinspirasi terminal CLI

**Color Philosophy**:
- Background: #0a0e17 (deep navy-black)
- Aksen utama: #00ff9d (neon green) — melambangkan transaksi sukses
- Aksen sekunder: #00cfff (electric cyan) — melambangkan data flow
- Warning: #ff6b35 (amber-orange) — untuk state rejected/expired

**Layout Paradigm**:
- Full-width dark canvas dengan panel-panel seperti terminal
- Sidebar kiri untuk navigasi dengan border neon
- Konten utama di kanan dengan grid data

**Signature Elements**:
1. Animated blockchain node graph di hero section
2. State machine diagram interaktif dengan transisi animasi
3. Code snippet blocks dengan syntax highlighting neon

**Interaction Philosophy**:
- Hover effects: border glow neon
- Click: ripple effect seperti transaksi on-chain
- Scroll: parallax dengan particle effect

**Animation**:
- Typing animation untuk tagline
- Particle network di background hero
- State machine transitions dengan smooth morph

**Typography System**:
- Display: Space Grotesk Bold
- Body: JetBrains Mono (monospace)
- Data: IBM Plex Mono

## Pendekatan 2: Brutalist Web3 — Bold & Architectural

**Design Movement**: Neo-Brutalism meets Blockchain Architecture

**Core Principles**:
1. Kontras ekstrem hitam-putih dengan satu warna aksen elektrik
2. Border tebal, shadow offset, elemen raw dan exposed
3. Layout non-konvensional — grid yang sengaja "rusak" tapi terstruktur
4. Teks besar, bold, dan berani sebagai elemen visual utama

**Color Philosophy**:
- Background: #f5f0e8 (warm cream/off-white)
- Foreground: #0f0f0f (near-black)
- Aksen: #5b21b6 (deep violet) — melambangkan smart contract authority
- Aksen 2: #fbbf24 (amber) — melambangkan nilai/aset yang di-escrow

**Layout Paradigm**:
- Asimetris dengan kolom berbeda lebar
- Elemen yang "menabrak" grid secara sengaja
- Section dividers dengan diagonal cuts

**Signature Elements**:
1. State machine diagram dengan gaya blueprint/schematic
2. Typography sebagai hero visual (teks besar yang menjadi gambar)
3. Card dengan border tebal dan shadow offset hitam

**Interaction Philosophy**:
- Hover: shadow offset bergeser (3D lift effect)
- Transisi: slide dan snap, bukan fade
- Cursor: custom crosshair

**Animation**:
- Entrance: elemen slide masuk dari sisi layar
- Hover: border dan shadow transform
- Scroll-triggered: counter animasi untuk statistik

**Typography System**:
- Display: Syne ExtraBold (800)
- Body: DM Sans Regular/Medium
- Code: Fira Code

## Pendekatan 3: Glassmorphism Cosmic — Premium Dark Web3

**Design Movement**: Glassmorphism meets Cosmic/Space Aesthetic

**Core Principles**:
1. Dark background dengan efek kaca frosted untuk card dan panel
2. Gradient warna kosmik (deep purple → midnight blue → teal)
3. Elemen mengambang dengan depth dan layering
4. Subtle grain texture untuk kedalaman visual

**Color Philosophy**:
- Background: oklch(0.08 0.02 270) (deep space)
- Glass panels: rgba(255,255,255,0.05) dengan backdrop-blur
- Aksen utama: oklch(0.75 0.18 200) (electric teal/cyan)
- Aksen sekunder: oklch(0.65 0.22 290) (violet/purple)
- Highlight: oklch(0.9 0.12 85) (golden amber)

**Layout Paradigm**:
- Full-viewport hero dengan cosmic background
- Floating card layout untuk fitur
- Horizontal scroll untuk timeline state machine
- Sticky navigation dengan glass blur

**Signature Elements**:
1. Animated orbital/network visualization di hero
2. Glass cards dengan gradient border
3. Glowing state indicators (Open=teal, Funded=blue, Submitted=amber, Completed=green, Rejected=red)

**Interaction Philosophy**:
- Hover: glass card lift dengan glow intensification
- Parallax: background stars/particles bergerak saat scroll
- Micro-animation: state badge pulse

**Animation**:
- Hero: floating orbs dengan slow drift animation
- Cards: staggered entrance dengan blur-in
- State machine: flow animation dari state ke state

**Typography System**:
- Display: Clash Display (bold, geometric)
- Body: Satoshi (clean, modern)
- Mono: JetBrains Mono
