# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | ✅ Yes    |

## Reporting a Vulnerability

If you discover a security vulnerability in AgentEscrow ERC-8183, please report it responsibly:

1. **Do NOT** open a public GitHub issue for security vulnerabilities
2. Contact us via X: [@_agentescrow](https://x.com/_agentescrow)
3. Include a detailed description of the vulnerability
4. We will respond within 48 hours and work with you on a fix

## Smart Contract Security

For vulnerabilities in the ERC-8183 smart contract implementation:
- Describe the attack vector clearly
- Provide a proof-of-concept if possible
- We will coordinate a responsible disclosure timeline

Thank you for helping keep AgentEscrow secure.
